## Default Permission

Default permissions for the rusqlite plugin.

#### This default permission set includes the following:

- `allow-load`
- `allow-execute`
- `allow-select`
- `allow-close`
- `allow-begin-transaction`
- `allow-commit-transaction`
- `allow-rollback-transaction`
- `allow-migrate`
